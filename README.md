# lit_python_toolbox

A consolidated repository for project-specific Python tools and research code developed by the *Laboratory of Imaging Technology (LIT)* at NHLBI.  
Each project is organized into its own branch for clarity and reproducibility.

---

## Branch Index (Contents)

Below is a summary of the currently active branches and what each contains. This section will grow as new projects are added.

### **1. `coil-sketching-4d`**  
**Topic:** Coil Sketching for 4D (3D+time) MRI Reconstruction  
**Description:**  
Implements an extended coil-sketching framework for memory-efficient 4D lung MRI reconstructions, including Motion-Compensated Low-Rank (MoCo-LR) solver scripts, Toeplitz approximation, and example Jupyter notebooks.  
**Branch link:**  
https://github.com/NHLBI/lit_python_toobox/tree/coil-sketching-4d
